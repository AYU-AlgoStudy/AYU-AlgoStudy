# AYU-AlgoStudy
기관 알고리즘 스터디 레포지토리

## 목표
- 최신 코딩테스트 빈출 BFS/DFS/트리/LCA 문제를 주 6문제 풀이
- 단순 정답이 아닌 **실수 원인 → 대응 패턴 → 정리**를 남김
- 쉽게 푼 문제는 제외, 틀린 문제/고민이 필요했던 문제 위주 공유

## 규칙
### 브랜치
- 개인은 **fork** 후, 매주 **주간 브랜치**에서 작업
- 브랜치명: `week01-닉네임` (예: `week01-zldzldzz`)
- 다음 주: `week02-닉네임` … (주 단위 새 브랜치)
- Org 레포 `main`에는 직접 push 금지, 반드시 PR로 반영

### 커밋 컨벤션 (Conventional Commits)
- 포맷: `type(scope): summary`
- 예시:
    - `feat(boj/2206): 벽 부수고 이동하기 BFS 3차원 visited 추가`
    - `fix(pgs/네트워크): 방문 배열 초기화 누락 수정`
    - `refactor(boj/7576): 큐→ArrayDeque 교체`
    - `docs(week01): 오답노트 정리`

### PR 규칙
- **제목**: `[week-01] 닉네임 — 4문제`
- **본문**: 문제/접근/실패 원인/교훈/참고링크 (템플릿 자동 적용)
- **머지 조건**: 최소 1인 승인 + CI 통과
- PR에는 코드 외에 **오답노트/블로그 링크(있으면)** 첨부

### 파일/디렉터리 규칙
- 닉네임/week주차.md (예 zldzldzz/week01.md)

## 운영 일정
- 기간: 2025-09-29 ~ 2025-11-07
- 주 6문제: 백준 + 프로그래머스 (난이도 실버~골드 초반)
- 풀이 일정: **월~금 + 일**, 토요일은 휴식
- 문제는 매일 카톡으로 공지
- 일요일: 한 주 회고/리뷰/정리
